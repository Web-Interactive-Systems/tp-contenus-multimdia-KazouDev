# 002 — Unity XR Students Project
Use Unity to build an educational VR scene. The scene should have

-   Multimodal contents: 3D Models, Images, Audios, Video, Texts
-   Navigation: Locomotion, Teleporation
-   Interaction
-   Affordance using color, audio, haptic
-   AI Agent(s) (using API/LM Studio or others) to do something useful for the end-user

Agent task = assists the end-user (e.g., a child, learner, adult) while they explore the experience, such as adding multimodal explanations, such as text and audio.

**The design question: What does the agent actually do for the end-user?**

For example (*agent as on-demand explainer*), user looks at an object, asks or click on a button *"what is that?"* — agent answers in context.
